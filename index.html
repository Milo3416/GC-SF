Introducción

CX Cloud: Genesys + Salesforce

Una integración nativa para voz, digital, routing inteligente, schema de datos común, agent copilot y gestión de fuerza laboral, lista para operar a escala empresarial.

01

Voz nativa y digital

Transfiera su operador u opte por Genesys Cloud Voice y canales digitales que aparecen de forma nativa en Salesforce, consolidando el trabajo del agente en una sola experiencia.

02

Esquema de datos común

Reduzca la carga de TI con datos sincronizados y compatibles con IA entre plataformas. Se elimina el mapeo frágil de atributos y se habilita una base consistente para automatización y personalización.

03

Enrutamiento inteligente

Genesys Cloud AI Predictive Routing asigna grandes volúmenes de interacciones a los agentes más adecuados, optimizando KPIs como AHT y FCR con datos históricos y en vivo.

04

Gestión de fuerza laboral

Pronóstico, capacidad, programación, tiempo libre y adherencia se gestionan en un modelo unificado para agentes de Salesforce, mejorando productividad y experiencia del empleado.

05

Listo para la empresa

Operación global con alta disponibilidad, cobertura multinacional y escala de voz empresarial, diseñada para soportar organizaciones con alta demanda y estándares operativos rigurosos.
